# Дипломный проект: Автоматизация тестирования Кинопоиска
Кинопоиск — крупнейший русскоязычный сервис с информацией о фильмах, сериалах и персонах. Проект автоматизирует проверки UI официального сайта и API неофициального сервиса.

## Задача проекта
Проект выполняет автоматизированную проверку ключевых сценариев работы сервиса.
Цель – убедиться в корректной работе поиска, фильтрации, навигации и воспроизведения контента.

### Покрываемые проверки
**UI-тесты (kinopoisk.ru)**

- Поиск фильмов по названию (параметризованный)
- Поиск фильмов по актёру
- Поиск фильмов по жанру
- Переход в раздел «Билеты в кино»
- Запуск трейлера со страницы фильма и проверка наличия видеоплеера в iframe

**API-тесты (api.poiskkino.dev)**

- Поиск фильмов по году и жанру
- Поиск фильмов по диапазону рейтинга IMDb
- Поиск фильмов по стране производства
- Получение фильма по ID
- Поиск персоны по имени

### Описание проекта

Проект реализован с использованием:
*   **Python** – язык программирования
*   **Pytest** – фреймворк для тестирования
*   **Selenium WebDriver** – для UI-тестов
*   **Requests** – для API-тестов
*   **Allure** – для построения отчетов

## Переменные окружения

Перед запуском создайте в корне файл **`.env`** со следующими ключами:

```ini
base_url = https://api.poiskkino.dev/
api_key = ваш_api_ключ
url_ui = https://www.kinopoisk.ru/
```
## Запуск тестов

### 1. Установка зависимостей

pip install -r requirements.txt

Содержимое requirements.txt:

pytest==9.0.2

selenium==4.18.1

requests==2.31.0

allure-pytest==2.13.2

python-dotenv==1.0.1

webdriver-manager==4.0.1

### 2. Запуск всех тестов.

pytest

### 3. Запуск только API-тестов

pytest -m api

### 4. Запуск только UI-тестов

pytest -m ui

### 5. Генерация Allure-отчёта

pytest --alluredir=allure-results
allure serve allure-results
